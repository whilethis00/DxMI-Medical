## 2. 무엇을 검증하나

**v1에서 확인된 문제 두 가지**:

1. **∇rw spike** — FM 샘플이 e_neg_std 폭발을 유발하는 step에서 최대 ∇rw=624 관찰. EBM gradient가 한 step에서 비정상적으로 크게 튀면서 학습 불안정.
2. **sep 고착 (sep=32.7)** — FM gate 개방 이후 `_sep_std_ema` 업데이트가 멈추는 버그. sep_std_ema가 gate 판정 후 freeze → 실제 분리도 변화를 반영하지 못함.

**이번 변경 (v1 → v2)**:

1. `energy_clamp: 20.0` — EBM 에너지 출력을 ±20 범위로 clamp.
   - 초기 고려값 6.0은 EBM 초기화 시 출력(-30~-50)이 전부 clamped → gradient 사망으로 기각.
   - 20.0은 정상 학습 구간(e±≈5)에선 거의 안 걸리고, spike(e_neg_std 폭발) 시점에만 개입.
2. `src/models/irl.py` sep monitoring 수정 — gate 개방 이후에도 `_sep_std_ema` 지속 업데이트.

**합격 기준**:
- best val Spearman ρ > v1 (0.2791)
- ep20 이후 ∇rw spike 없음 (후반 ∇rw ≤ 20 기준)
- sep_std_ema가 gate 이후에도 가변적으로 유지 (v1의 32.7 고착 해소)

## 5. 결과 해석 및 인사이트

### 긍정적

- **∇rw spike 방어**: ep20 이후 ∇rw 대부분 < 10. v1의 최대 624에서 ep30 기준 5~9로 수렴. energy_clamp=20.0이 정상 학습 구간에서 이상 gradient를 효과적으로 차단.
- **sep monitoring fix 효과 확인**: ep30 sep=38→74→65 가변적 유지. v1의 sep=32.7 고착 해소. gate 이후에도 실제 분리도 변화를 추적.
- **best val ρ=0.2848 (ep29)**: p=0.0001로 강한 유의성. v1(0.2791) 대비 개선. ep16 이후 FAIL은 ep20 단발뿐이며 ep21~30은 9/10 PASS.
- **fm_e 수렴 (ep30 ≈+5.0)**: FM 샘플 에너지가 SGLD neg(e-≈+5)와 동등 수준. FM 샘플이 의미 있는 hard negative로 정착됨을 시사.
- **점진적 ρ 상승 (ep25~29)**: ρ=0.2306 → 0.2514 → 0.2170 → 0.2657 → 0.2848 — 후반 상승 추세. SGLD-only 실험(ep14~20 이후 정체 or 하락)과 다른 패턴. FM hard negative가 후반 ranking 개선에 기여하는 증거.

### 주의

- **초기 FM spike (ep02~03)**: gate 개방 직후 ∇rw=240~712. energy_clamp=20.0도 초기 distribution collapse(e_pos_std=±3) 구간엔 효과 제한적. 20.0 clamp가 실제 개입하는 구간은 e±가 ±20을 넘을 때인데, 초기엔 더 심하게 벗어나며 오히려 clamp 자체가 gradient를 죽이는 방향.
- **ep12~15 불안정**: 첫 PASS(ep12, ρ=0.2218) 이후 ep13~15 연속 FAIL. v1과 동일 패턴. FM 도입 직후 EBM이 새로운 negative 분포에 적응하는 전환 구간으로 해석.
- **ep14 ECE 이상 (ECE=0.3846)**: 정상 범위(~0.79)와 크게 다름. 동시에 ρ=-0.0923, AUROC=0.3943. 이 epoch에서 EBM의 energy scale이 일시적으로 왜곡된 것으로 보임. FM 샘플과 SGLD 샘플 간 에너지 충돌로 인한 CD loss 이상 가능성.
- **ep20 단발 FAIL (ρ=0.1149)**: ep19(ρ=0.2249) 직후 급락. 이후 ep21부터 회복. v1도 유사한 단발 FAIL 관찰 → FM hybrid의 구조적 불안정성 잔존 (spike가 아닌 분포 변화 원인일 가능성).
- **seed=1 단일 run**: v1과 마찬가지로 3-seed 재현 없음. 현재 수치는 seed-specific일 수 있으며 우위 주장을 위해 seed 2, 3 재현 필수.
- **ECE 고착 (≈0.79)**: energy_clamp가 ranking에는 도움이 됐으나 calibration 문제는 미해결. B(ECE=0.232) 대비 ~0.56 격차는 구조적 문제(CD loss 특성).

### 근본 원인 가설

- **ep14 ECE 이상**: FM 샘플이 energy_clamp 범위를 일시적으로 넘어서며 CD loss 분자/분모가 비정상적으로 계산됨 → calibration 왜곡. 또는 SGLD replay buffer에 오래된 (이제는 easy한) negative가 섞이면서 EBM이 과도한 separation 시도 후 조정.
- **초기 spike vs 후반 안정**: FM 샘플의 에너지가 초기엔 EBM 범위 밖(-1~+6.8 혼재)이다가 ep06 이후 +4~+6으로 수렴. 이 수렴 이후 energy_clamp가 실질적으로 작동하기 시작 → ∇rw 안정화.
- **ECE vs ranking 분리**: CD loss는 binary ranking (real vs neg)을 최적화하며 probability calibration을 목표로 하지 않음. Temperature scaling이 필요한 구조적 이유.
